# CopyTrack 1.0.0

## Highlights

- First public macOS release
- Local clipboard history for text, links, images, and file references
- Quick Access popup with editable global shortcut
- Menu bar workflow with the latest four copied items
- First-run onboarding inside the app
- Favorites, pins, tags, filters, sorting, import, and export
- English and Russian interface support

## Install

1. Download the macOS `.dmg` asset from this release.
2. Open the disk image.
3. Drag `CopyTrack.app` into `Applications`.
4. Launch CopyTrack and allow clipboard access if macOS asks.

## Notes

- CopyTrack is local-first and does not require an account.
- Signing and notarization are not configured yet.
- The initial public release targets macOS first.

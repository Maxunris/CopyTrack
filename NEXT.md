# NEXT

This file collects ideas that are intentionally out of scope for version 1.0 but are worth preserving for later iterations.

## Product Expansion

- Cloud sync across devices with optional account sign-in
- End-to-end encryption for synced clipboard history
- Local vault mode for sensitive entries protected by password or biometrics
- Shared collections or team snippets for collaborative workflows
- OCR for images copied into history
- Smart snippet suggestions and duplicate grouping
- Richer clipboard history import/export formats beyond the current JSON snapshot flow

## Platform Expansion

- Windows support with native-feeling tray and startup integration
- Linux support with desktop-environment-specific clipboard adapters
- Cross-platform packaging and release automation for macOS, Windows, and Linux

## UX Ideas

- Pinned collections for reusable snippets
- Spotlight-style command mode for actions on clipboard items
- Rich preview cards for code, links, and images
- Optional onboarding tips and first-run permission assistant
